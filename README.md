# 📦 Dashboard Logístico Estratégico

Projeto de dashboard logístico desenvolvido com foco em visão gerencial da operação, centralização de indicadores operacionais e automação de análises.

---

# 🎯 Objetivo do Projeto

O projeto foi desenvolvido a partir de uma necessidade real do negócio: consolidar informações operacionais de parceiros logísticos em uma visão executiva clara e dinâmica.

A proposta foi transformar uma base operacional extensa e descentralizada em indicadores estratégicos para apoio à tomada de decisão.

---

# 📊 Indicadores Monitorados

- Total de NF expedidas
- Faturamento total
- Volume total transportado
- Média de volume por NF
- Performance por Centro de Distribuição (CD)
- Status de entrega
- SLA operacional
- Expedição por período
- Análise por projeto

---

# ⚙️ Funcionalidades

✅ Atualização automatizada da base  
✅ Consolidação operacional  
✅ Visão executiva logística  
✅ Indicadores gerenciais  
✅ Segmentações dinâmicas  
✅ Comparativos operacionais  
✅ Análise por CD e projeto  

---

# 🚀 Automação da Base

A estrutura foi desenvolvida para permitir atualização automática dos indicadores através da inserção de uma nova extração do TMS na base principal.

Com isso, o processo reduz atividades manuais e agiliza o acompanhamento operacional.

---

# 🛠️ Tecnologias Utilizadas

- Excel
- Power Query
- Tabela Dinâmica
- Modelagem de Dados
- Indicadores Operacionais
